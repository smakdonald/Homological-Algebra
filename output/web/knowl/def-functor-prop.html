Definition 1.2.7.

Definition 1.29. A covariant functor \(F: \mathscr{C} \longrightarrow \mathscr{D}\) between locally small categories is
  • faithful if all the functions of sets
are injective.
\begin{equation*} \begin{gathered} \operatorname{Hom}_{\mathscr{C}}(A, B) \longrightarrow \operatorname{Hom}_{\mathscr{D}}(F(A), F(B)) \\ f \longmapsto F(f) \end{gathered} \end{equation*}
  • full if all the functions of sets
\begin{equation*} \begin{gathered} \operatorname{Hom}_{\mathscr{C}}(A, B) \longrightarrow \operatorname{Hom}_{\mathscr{D}}(F(A), F(B)) \\ f \longmapsto F(f) \end{gathered} \end{equation*}
are surjective.
  • fully faithful if it is full and faithful.
  • essentially surjective if every object \(d\) in \(\mathscr{D}\) is isomorphic to \(F c\) for \(c\) in \(\mathscr{C}\text{.}\)
  • an embedding if it is fully faithful and injective on objects.
