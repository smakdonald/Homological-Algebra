Example 1.1.16.

  1. The empty set is initial in Set. Any singleton is terminal. Since the empty set and a singleton are not isomorphic in Set, there is no zero object in Set.
  2. The 0 module is the zero object in \(R\)-mod.
  3. The trivial group \(\{e\}\) is the zero object in Grp.
  4. In the category of rings, \(\mathbb{Z}\) is the initial object, but there is no terminal object unless we allow the 0 ring.
  5. There are no initial nor terminal objects in the category of fields.
