Example 4.41. However, it is not flat. For example, when we apply \(R / \mathfrak{m} \otimes_{R}-\) to the inclusion \(\mathfrak{m} \subseteq R\) we obtain a map of \(R / \mathfrak{m}\)-vector spaces
\begin{equation*} \mathfrak{m} / \mathfrak{m}^{2} \longrightarrow R / \mathfrak{m} \end{equation*}
