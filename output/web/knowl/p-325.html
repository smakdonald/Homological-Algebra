If \(a^{\prime} \in \operatorname{ker} f\text{,}\) then
\begin{equation*} g\left(i^{\prime}\left(a^{\prime}\right)\right)=i f\left(a^{\prime}\right)=0 \end{equation*}
by commutativity, so \(i^{\prime}\left(a^{\prime}\right) \in \operatorname{ker} g\text{.}\) Similarly, if \(b^{\prime} \in \operatorname{ker} g\) then \(p^{\prime}\left(b^{\prime}\right) \in \operatorname{ker}(g)\text{.}\) So
\begin{equation*} A^{\prime} \stackrel{i^{\prime}}{\longrightarrow} B^{\prime} \stackrel{p^{\prime}}{\longrightarrow} C^{\prime} \quad \text { restrict to maps } \quad \operatorname{ker} f \stackrel{i^{\prime}}{\longrightarrow} \operatorname{ker} g \stackrel{p^{\prime}}{\longrightarrow} \operatorname{ker} h \text {. } \end{equation*}
