Proof

  1. As a function defined on each fixed \(\operatorname{Hom}_{R}(M, N), T\) is a group homomorphism, so it must send \(0\) to \(0\text{.}\)
  2. An \(R\)-module \(M\) is the zero module if and only if the zero and identity maps on \(M\) coincide. Let \(N\) be the image of the zero \(R\)-module via \(T\text{.}\) On the one hand, any functor must send identity maps to identity maps, so the identity map on the zero module must be sent to the identity on \(N\text{.}\) On the other hand, we have shown that the zero map must be sent to the zero map on \(N\text{,}\) so the zero and identity maps on \(N\) must coincide, so \(N=0\text{.}\)
