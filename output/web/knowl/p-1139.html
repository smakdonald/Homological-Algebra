A ring is a set \(R\) equipped with two binary operations, + and \(\cdot\text{,}\) satisfying:
  1. \((R,+)\) is an abelian group with identity element denoted 0 or \(0_{R}\text{.}\)
  2. The operation \(\cdot\) is associative, so that \((R, \cdot)\) is a semigroup.
  3. For all \(a, b, c \in R\text{,}\) we have
    \begin{equation*} a \cdot(b+c)=a \cdot b+a \cdot c \quad \text { and } \quad(a+b) \cdot c=a \cdot c+b \cdot c \end{equation*}
  4. there is a multiplicative identity, written as 1 or \(1_{R}\text{,}\) such that \(1 \neq 0\) and \(1 \cdot a=a=a \cdot 1\) for all \(a \in R\text{.}\)
