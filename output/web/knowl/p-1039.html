If \(E\) is injective, it is also divisible, by Lemma 4.22. Given any submodule \(D \subseteq E\text{,}\) any \(e \in E\text{,}\) and a nonzero \(r \in R\text{,}\) there exists \(y \in E\) such that \(r y=e\text{,}\) and so this also holds in \(E / D\text{.}\) Then \(E / D\) is divisible, and thus injective by Lemma 4.25.  1
