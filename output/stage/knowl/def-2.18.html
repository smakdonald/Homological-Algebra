Definition 2.25.

Definition 2.18. We say that a short exact sequence
\begin{equation*} 0 \longrightarrow A \longrightarrow B \longrightarrow C \longrightarrow 0 \end{equation*}
splits or is a split short exact sequence if it is isomorphic to
\begin{equation*} 0 \longrightarrow A \stackrel{i}{\longrightarrow} A \oplus C \stackrel{p}{\longrightarrow} C \longrightarrow 0 \end{equation*}
where \(i\) is the inclusion of the first component and \(p\) is the projection onto the second component.
