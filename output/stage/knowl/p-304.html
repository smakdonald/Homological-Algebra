Example 2.22. The short exact sequence
\begin{equation*} 0 \longrightarrow \mathbb{Z} \stackrel{2}{\longrightarrow} \mathbb{Z} \longrightarrow \mathbb{Z} / 2 \longrightarrow 0 \end{equation*}
is not split. Indeed, \(\mathbb{Z}\) does not have any 2 -torsion elements, so it is not isomorphic to \(\mathbb{Z} \oplus \mathbb{Z} / 2\text{.}\)
