Example 4.12. Projective but not Free Module.

Example 4.12. The ring \(R=\mathbb{Z} /(6)\) can be written as a direct sum of the ideals
\begin{equation*} I=(2) \text { and } J=(3) \end{equation*}
Indeed, \(R=I+J\) and \(I \cap J=0\text{,}\) so \(R=I \oplus J\text{.}\) By CorollaryÂ 4.11, \(I\) and \(J\) are projective \(R\)-modules. However, \(I\) and \(J\) are not free. This can be explained numerically: every finitely generated free \(R\)-module is of the form \(R^{n}\text{,}\) so it has \(6^{n}\) elements for some \(n\text{,}\) while \(I\) and \(J\) have \(3\) and \(2\) elements respectively.
