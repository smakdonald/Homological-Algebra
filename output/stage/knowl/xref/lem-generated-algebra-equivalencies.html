Lemma 8.18. Generated Algebra Equivalencies.

The following are equivalent:
  1. \(\Lambda\) generates \(S\) as an \(R\)-algebra.
  2. Every element in \(S\) admits a polynomial expression in \(\Lambda\) with coefficients in \(\phi(R)\text{,}\) i.e.
    \begin{equation*} S = \left\{\sum_{\mathrm{finite}} \phi(r_{i_1, \ldots, i_n}) \lambda_1^{i_1} \cdots \lambda_n^{i_n} \mid r_l \in R, \lambda_j\in\Lambda, i_j \geqslant 0 \right\}. \end{equation*}
  3. If \(R[X]\) is a polynomial ring on a set of indeterminates \(X\) in bijection with \(\Lambda\text{,}\) then the \(R\)-algebra homomorphism
    \begin{equation*} \begin{CD} R[X]^{n}@>\pi>> S\\\\ x_{i}@>>> \l_{i} \end{CD} \end{equation*}
    is surjective.

Proof.

Let \(S= \left\{\sum_{\mathrm{finite}} \phi(a) \lambda_1^{i_1} \cdots \lambda_n^{i_n} \mid a\in A, \lambda_j\in\Lambda, i_j\in \N\right\}\text{.}\)
For the equivalence between (2) and (3), we note that \(S\) is the image of \(\pi\text{.}\) In particular, \(S\) is a subring of \(R\text{.}\) It then follows from the definition that (1) implies (2). Conversely, any subring of \(R\) containing \(\phi(A)\) and \(\Lambda\) certainly must contain \(S\text{,}\) so (2) implies (1).
