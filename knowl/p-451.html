An additive functor \(T: R\)-Mod \(\longrightarrow S\)-Mod is an exact functor if it preserves short exact sequences. When \(T\) is covariant, this means that every short exact sequence
\begin{equation*} 0 \longrightarrow A \stackrel{f}{\longrightarrow} B \stackrel{g}{\longrightarrow} C \longrightarrow 0 \end{equation*}
is taken to the short exact sequence
\begin{equation*} 0 \longrightarrow T(A) \stackrel{T(f)}{\longrightarrow} T(B) \stackrel{T(g)}{\longrightarrow} T(C) \longrightarrow 0 \end{equation*}
