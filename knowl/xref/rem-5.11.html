Proposition 4.14.

The homology of a direct sum is the direct sum of the homologies.

Proof.

Notice
\begin{equation*} \left(\partial_{n}, \delta_{n}\right)(a, b)=(0,0) \Longleftrightarrow \partial_{n}(a)=0 \text { and } \delta_{n}(b)=0 \text {, } \end{equation*}
and
\begin{equation*} (a, b) \in \operatorname{im}\left(\partial_{n}, \delta_{n}\right) \text { if and only if } a \in \operatorname{im} \partial_{n} \text { and } b \in \operatorname{im} \partial_{n} \end{equation*}
Thus
\begin{equation*} \mathrm{H}_{n}(F \oplus G)=\frac{\operatorname{ker}\left(\partial_{n}, \delta_{n}\right)}{\operatorname{im}\left(\partial_{n+1}, \delta_{n+1}\right)}=\frac{\operatorname{ker} \partial_{n}}{\operatorname{im} \partial_{n+1}} \oplus \frac{\operatorname{ker} \delta_{n}}{\operatorname{im} \delta_{n+1}}=\mathrm{H}_{n}(F) \oplus \mathrm{H}_{n}(G) \end{equation*}
